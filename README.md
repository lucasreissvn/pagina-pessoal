# Página Pessoal GAC-116 Programação Web

Projeto desenvolvido para a disciplina **GAC-116 Programação Web**, com foco na criação de uma página pessoal utilizando frameworks de front-end.

## Acesso Online

Acesse o projeto publicado:
https://lucasreissvn.github.io/pagina-pessoal/

## Execução Local

Siga os passos abaixo para rodar o projeto localmente:

```bash
git clone https://github.com/lucasreissvn/pagina-pessoal.git
cd pagina-pessoal
```

Depois, basta abrir o arquivo `index.html` no seu navegador.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap
* W3.CSS

## Objetivo do Projeto

Desenvolver uma página pessoal responsiva aplicando conceitos fundamentais de desenvolvimento web, incluindo:

* Estruturação com HTML
* Estilização com CSS
* Uso de frameworks para layout e responsividade
<br>

> **Nota:** Este projeto foi desenvolvido exclusivamente para fins acadêmicos.
